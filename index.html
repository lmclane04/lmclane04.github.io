Hi, I'm Lauren.

I'm a Stanford undergrad that likes to make things.
Here are a couple projects I've worked on.

  • LED Tetris!
    Wired, soldered, and programmed Tetris on a PCB using an Arduino Uno and LED multiplexing.
  • Shy (and angry) Useless Box
    Simple Useless box FSM upgraded with an Arduino Uno for different behavior patterns.
  • Harmonic Music Player and Display
    Sophisticated music player with harmonics, ADSR-envelope dynamics, reverb, stereo panning, and a realtime waveform display using double buffering on a PYNQ-Z1 FPGA.
  • Password Hasher
    Implemented Brownie password hashing algorithm and checker on a PYNQ-Z1 board.
  • Floating Point Adder
    Simple floating point adder on a PYNQ-Z1.
  • Bike Light
    Adjustable blinking bike light FSM on a PYNQ-Z1.
